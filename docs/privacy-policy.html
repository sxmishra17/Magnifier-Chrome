Privacy Policy

Magnifier - Hover Zoom Lens for Chrome · Last updated: March 31, 2026
No data collected

Overview

Magnifier is a browser extension that provides a hover-activated zoom lens for webpages. This privacy policy explains what data the extension does and does not collect.

Data We Do Not Collect

Magnifier does not collect, store, transmit, or share any of the following:

  • Browsing history or visited URLs
  • Page content, text, or images from any website
  • Personal information of any kind
  • Keystroke data or mouse movement data
  • Cookies or authentication tokens
  • Device identifiers or IP addresses
  • Crash reports or usage analytics

Data We Do Store (Locally Only)

The extension saves the following user preferences using Chrome's built-in sync storage API:

  • Zoom level (a number between 1 and 10)
  • Lens size (small, medium, or large)
  • Lens shape (circle or rectangle)
  • Lens position (left, right, top, bottom, or centre)

These preferences are stored locally in your Chrome profile via chrome.storage.sync. They may sync across your own devices if you are signed into Chrome sync. They are never sent to any external server, and we have no access to them.

No Third Parties

Magnifier does not use any third-party analytics services, advertising networks, tracking pixels, or SDKs. There are no external network requests of any kind made by the extension.

PDF Files

The built-in PDF viewer opens files selected by the user directly in the browser using a local object URL. PDF files are never uploaded, transmitted, or stored anywhere outside the user's device. They are discarded from memory when the viewer is closed or cleared.

Permissions

The extension requests the following Chrome permissions, each used only for the stated purpose:

  • storage - to save and restore your magnifier settings locally
  • activeTab - to identify the current tab so the popup can control the magnifier for that tab only
  • scripting - to inject the magnifier lens overlay into the active tab when you enable it
  • tabs - to send settings updates from the popup to the content script in the active tab
  • host permissions (<all_urls>) - so the lens can be activated on any website the user chooses

Children's Privacy

Magnifier does not collect any data from anyone, including children. It is safe for use by users of all ages.

Changes to This Policy

If this policy is updated, the updated version will be published at this URL with a revised date. Continued use of the extension constitutes acceptance of any changes.

Contact

If you have any questions about this privacy policy, you may contact us via the Chrome Web Store support page for the Magnifier extension.
